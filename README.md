# Консольна утиліта «Каталог фільмів» (Варіант №4)

Проєкт виконано в межах навчальної практики з розробки програмного забезпечення.
**Виконавець:** Бондарук Владислав, студент групи ІПЗ-21.

## Функціонал програми
- Додавання фільмів із валідацією року та текстових полів.
- Перегляд усього каталогу у вигляді акуратної консольної таблиці.
- Регістронезалежний пошук (фільтрація) за жанрами чи статусом.
- Оцінювання фільмів від 1 до 10 з автоматичною зміною статусу на "Переглянуто".
- Видалення фільмів за порядковим номером.
- Автоматичне збереження даних у локальний файл `movies.json`.

2. Відкрийте термінал або командний рядок у цій папці.
